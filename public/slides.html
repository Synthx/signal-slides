React to changes: Effect

React to changes: Effect

Effects are callback functions that are triggered when the signals they depend on change.

Used for DOM manipulations, data fetching, and subscriptions.

React to changes: Effect

                    
                        const [count, setCount] = createSignal(0);

                        createEffect(() => {
                            // will run every time count changes
                            console.log(count());
                        });

                        const increment = () => {
                            setCount((prev) => prev + 1);
                        };

Compose signals

Compose signals

Composed / Derived / Memo signals are pure functions that rely on one or more signals to produce a derived value.

Compose signals

                    
                        const [count, setCount] = createSignal(0);

Compose signals

                    
                        const [count, setCount] = createSignal(0);

                        const doubledCount = createMemo(() => count() * 2);

                        console.log(doubledCount()); // output: 0

                        setCount(3);

                        console.log(doubledCount()); // output: 6
